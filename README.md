# Agfresh App

Starter Vite + React + TailwindCSS project.

## Setup

Install dependencies:

```powershell
npm install
```

Run dev server:

```powershell
npm run dev
```

Build for production:

```powershell
npm run build
```

Preview production build:

```powershell
npm run preview
```
